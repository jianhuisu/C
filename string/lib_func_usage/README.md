### 注意事项
  
Man Page的BUGS部分指出了用strtok和strtok_r函数需要注意的问题：

 - 这两个函数要改写字符串以达到分割的效果
 - 这两个函数不能用于常量字符串，因为试图改写.rodata段会产生段错误
 - 在做了分割之后，字符串中的分隔符就被'\0'覆盖了
 - strtok函数使用了静态变量，它不是线程安全的，必要时应该用可重入的strtok_r函数，以后再详细介绍“可重入”和“线程安全”这两个概念
